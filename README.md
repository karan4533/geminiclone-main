## ⚙️ How to Run

1. **Clone the repo**

```bash
git clone https://github.com/karan4533/geminiclone-main.git
````

2. **Go to the project folder**

```bash
cd geminiclone-main
```

3. **Install dependencies**

```bash
npm install
```

4. **Start the app**

```bash
npm run dev
```

Now open [http://localhost:3000](http://localhost:3000) in your browser.
